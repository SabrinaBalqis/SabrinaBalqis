Hi, I'm **Sabrina** — a Software Engineering graduate (Hons.) who bridges design, development, and business strategy.

## Featured Projects

**Questify** — Gamified To-Do List
> Daily tasks turned into RPG-style quests with XP, level-ups, character progression, and weather-aware quest suggestions via a live API. Full auth + real-time Supabase sync.
`Angular` `TypeScript` `TailwindCSS` `Supabase` `OpenWeather API`

**MoodFood** — Mood-Driven Recipe Roulette
> A custom Angular spin-wheel component that recommends meals based on mood, cuisine, diet, and calorie preferences — pulling live data from two external recipe APIs.
`Angular` `TypeScript` `TailwindCSS` `Spoonacular API` `TheMealDB API`

**WanderList** — Travel Dashboard & Budget Tracker
> Full trip-planning dashboard with day-by-day itineraries, checklists, and an automated budget calculator running on full CRUD + real-time Supabase data.
`Angular` `TypeScript` `TailwindCSS` `Supabase`

**Expenses Splitter** — Full-Stack Group Expense Tracker
> Multi-group expense-sharing app with an automated calculation engine for transaction breakdowns and peer-to-peer debts. Custom REST API backend with real-time cloud sync.
`React` `Redux Toolkit` `Node.js` `Express` `Supabase`

**Apex Academy** — Corporate Platform Redesign
> Rebuilt page layouts, navigation, and component styling for a live e-learning platform used daily by Apex Equity staff — shipped to production, not just a mockup.
`PHP` `TailwindCSS` `Docker`

**FloralPetals** — Bouquet Customisation System
> Full-stack e-commerce platform for personalised flower orders with a live price preview, real-time inventory, order tracking, and a 3NF-normalised database on MVC architecture.
`Java` `JSP/Servlet` `MySQL`
